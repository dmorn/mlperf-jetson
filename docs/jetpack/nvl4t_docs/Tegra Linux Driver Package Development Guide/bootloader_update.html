NVIDIA® Jetson™ Linux Driver Package supports Bootloader update and redundancy on all Jetson platforms. However, the implementations of update and redundancy on NVIDIA® Jetson Nano™ and Jetson TX1 platforms are substantially different than on NVIDIA® Jetson Xavier™ series and NVIDIA® Jetson™ TX2 series platforms.
