This topic describes window systems supported in L4T:
• Gnome (the standard Ubuntu window system, the default window system on most NVIDIA® Jetson™ platforms)
• Gnome with Wayland (a newer server communication protocol designed as a replacement for X Window System)
• X server (the standard implementation of the X Window System, also known as X11)
• LDXE (a lightweight X11 window system, the default window system on Jetson Nano 2G)
