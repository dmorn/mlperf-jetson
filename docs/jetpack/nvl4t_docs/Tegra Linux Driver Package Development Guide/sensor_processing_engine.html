Sensor Processing Engine (SPE)

NVIDIA® Jetson™ products provide a built-in Cortex-R5 microcontroller within an always-on power domain which is known as the Sensor Processing Engine (SPE). SPE is useful for sensor data processing, wake up management, and robotics. It runs with software based on FreeRTOS.
For detailed information about SPE, see the Jetson Sensor Processing Engine (SPE) Developer Guide and source files and the technical reference manual for your Jetson platform, all provided in the Jetson Download Center.
