This topic describes security features of NVIDIA® Jetson™ Linux Driver Package (L4T). It has two subtopics:
• Secureboot describes Secureboot, a feature which ensures that the Jetson Linux boot process cannot be redirected or compromised.
• Trusty, a feature which provides a secure environment (a Trusted OS, or TOS) in which applications can run.
