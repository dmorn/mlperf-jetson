This part of the Bootloader topic describes the boot flow.
The primary function of the Jetson boot software is to initialize the SoC (System on a Chip), including:
• Initializing MC/EMC/CPU
• Setting up security parameters
• Loading different firmware
• Maintaining Chain of Trust
• Setting memory carveouts for different firmware
• Flashing the device
• Booting to the operating system
Additionally, the Jetson boot software also performs other operations defined by product requirements, including but not limited to:
• Initialization of HDMI™/DP
• Displaying the boot logo
